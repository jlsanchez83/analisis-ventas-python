# 📊 Análisis de ventas con Python

## 🧠 Objetivo
Este proyecto tiene como objetivo analizar datos de ventas para identificar productos más vendidos, ingresos por categoría y tendencias mensuales.

---

## 🛠️ Herramientas utilizadas
- Python
- pandas
- matplotlib
- Google Colab

---

## 📊 Análisis realizado
- Lectura de datos desde archivo CSV
- Conversión de fechas
- Cálculo de ventas totales
- Análisis por producto
- Análisis por categoría
- Análisis por región
- Análisis mensual de ventas
- Visualización de datos con gráficos
- Exportación de resultados a Excel

---

## 📈 Resultados principales
- El producto con mayor ingreso fue **Laptop**
- La categoría con mayores ventas fue **Technology**
- Las ventas muestran crecimiento entre enero y febrero
- Lima concentra la mayor cantidad de ventas

---

## 📁 Archivos del proyecto
- `analisis.ipynb` → Notebook con el análisis
- `ventas.csv` → Dataset utilizado
- `reporte_ventas.xlsx` → Reporte generado
- `README.md` → Descripción del proyecto

---

## 📌 Conclusión
Este análisis permite entender el comportamiento de ventas y puede servir como base para la toma de decisiones en un negocio.
